Apache Security Mailing List

Connect with the Apache Security community through our mailing lists

Private Security Reports

To privately disclose a security vulnerability in one of our projects, send your report to:

  • security@apache.org

The Apache Security Team will work with you to resolve the vulnerability.

Public Security Discussions

To discuss ASF-wide security-related topics, join the public security-discuss mailing list:

  • Read the archives
  • Subscribe by emailing security-discuss-subscribe@community.apache.org
  • Once subscribed, send your mail to security-discuss@community.apache.org

Project-Specific Security Lists

Some Apache projects maintain their own security mailing lists. Check the individual project pages for more information.

If a project doesn't have a dedicated security contact, use the main security@apache.org address.
